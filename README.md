# 🌐 PolNews AI Bot

Умный Telegram-бот с AI-аналитиком для политических и экономических новостей.
**Языки:** Русский 🇷🇺 | O'zbek 🇺🇿

---

## 🚀 Быстрый старт

### 1. Установи зависимости
```bash
pip install -r requirements.txt
```

### 2. Создай бота в Telegram
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Придумай имя и username (например `@PolNewsAIBot`)
4. Скопируй токен

### 3. Получи Anthropic API ключ
1. Зайди на [console.anthropic.com](https://console.anthropic.com)
2. API Keys → Create Key
3. Скопируй ключ

### 4. Настрой .env файл
```
BOT_TOKEN=твой_токен_от_botfather
ANTHROPIC_API_KEY=твой_anthropic_ключ
```

### 5. Запусти бота
```bash
python bot.py
```

## 📂 Структура проекта
```
news_bot/
├── bot.py           # Главный файл — хендлеры, логика
├── config.py        # RSS-ленты и настройки
├── database.py      # Хранение пользователей и статей (JSON)
├── texts.py         # Все тексты на RU и UZ
├── requirements.txt
├── .env             # Твои ключи (не публикуй!)
└── data.json        # Создаётся автоматически
```

## 🤖 Функции AI (Claude)

| Функция | Описание |
|---------|----------|
| **AI Анализ статьи** | Разбирает любую новость: влияние, последствия, оценка |
| **AI Чат** | Диалог с экспертом по политике/экономике с памятью контекста |
| **Дайджест дня** | AI собирает и структурирует главные события дня |
| **Тренды** | AI определяет горячие темы по заголовкам |
| **Умный поиск** | AI отвечает на вопросы по любой теме |
